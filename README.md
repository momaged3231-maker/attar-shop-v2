# Attar Shop V2

واجهة متجر عربية فاخرة لنشاط عطارة، مبنية كـ UI demo جاهز للعرض على العملاء.

## Features

- RTL Arabic interface.
- Modern Luxury visual direction.
- Product category filters.
- 16 fake products with prices and descriptions.
- Add to cart interaction with animated sidebar.
- Favorite products UI.
- Testimonials carousel.
- Newsletter UI section.
- Smooth Framer Motion animations.
- Fully responsive layout.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Notes

This is a UI-only project. There is no backend, no checkout, and no real payment flow.
